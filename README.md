# bdix_tester
Find your nearby bdix server

```
pip install -r requirements.txt 
```

### create virtual env: 
setup virtualenv
```
pip3 install virtualenv 
virtualenv venv
```
Active your virtual environment:    
```
source venv/bin/activate
```

To deactivate:
```
deactivate
```

### Instead of using virtualenv you can use this command in Python3
```
python3 -m venv venv
```
